


# Spam/Not Spam Classification Website
Overview

This project is a web-based application that classifies messages as either "Spam" or "Not Spam" using machine learning models. The goal is to provide a simple and efficient tool for users to identify and filter out spam messages.

# Features
Machine Learning Models: Multiple models were tested, including Logistic Regression, Random Forest, Support Vector Machines, and Extra Tree Classifier. The model with the best performance was selected for deployment.

User-Friendly Interface: The frontend is designed with simplicity in mind, allowing users to easily input a message and receive an instant classification.
Fast Predictions: The selected model provides quick and accurate spam detection, making it practical for everyday use.

# Technology Stack

**Frontend**: HTML, CSS, Streamlit  
**Backend**: Python, Scikit-learn, Streamlit  
**Modeling**: Various machine learning classifiers, with Extra Tree Classifier selected for final deployment
